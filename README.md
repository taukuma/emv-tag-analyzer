## EMV Tag Decoder
"EMV Tag Decoder" decodes EMV tags and values in Tag-Length-Value (TLV) format based on EMV Book 3 and Book 4 (and Book C series for contactless payment).

### How To Use
```
var data = TLV.Analyze("TLV DATA");
console.log(data);
```
